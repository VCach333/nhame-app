# Nhame APP

**An APP to save and manage your foods**


## How it work?

*Create Ingredients and Spice. Use it to create your Souces, Side dish and Main dish. Finaly, use the Souces, Side dish and Main dish the create your Dish. The APP can make a list for your dish weekly or monthly. You can generate a list always*



# How to use?

## Downloading Source Code & Run it on Expo Emulator

### Requirements: 

*NodeJS installed (www.nodejs.org)*
*Expo APP (get it on the Play Store or Apple Store)*
*An Android/IOS Emulator or a WLAN, PAN USB Connection or other way to connect your computer with your Smartphone*


### Downlaod and Run the Project

*(1) run `git clone https://github.com/VCach333/nhame-app.git` and it will download a compressed folder (like .zip)*
*(2) extract the compressed folder*
*(3) on your computer open the terminal and navigate to folder extrated*
*(4) run `npm install` and `npx expo install` (if it require any tool accept)*
*(5) run `npx expo start` (if it require any tool accept)*
*(6) Using Expo APP scan the QRCode showed on terminal with your smartphone or type the IP address with the PORT (eg.: 192.168.10.10:8080)*
