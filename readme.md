# Nhame APP

**An APP to save and manage your foods**


## How it work?

*Create Ingredients and Spices. Use it to create your Souces, Side dishes and Main dishes. Finaly, use the Souces, Side dishes and Main dishes the create your Dishes. The APP can make a list for your dishes weekly or monthly. You can generate a list always*


## How to use?

### Downloading Source Code & Run it on Expo Emulator

#### Requirements: 

*NodeJS installed (www.nodejs.org)<br>*
*Expo APP (get it on the Play Store or Apple Store)<br>*
*An Android/IOS Emulator or a WLAN, PAN USB Connection or other way to connect your computer with your Smartphone*


### Downlaod and Run the Project

*1- run `git clone https://github.com/VCach333/nhame-app.git`*
*  1- it will download a compressed folder (like .zip)*
*  1- extract the compressed folder*
*1- on your computer open the terminal and navigate to folder extrated*
*1- run `npm install` and `npx expo install`*
*  1- if it require any tool accept*
*1- run `npx expo start`*
*  1- if it require any tool accept*
*1- Using Expo APP*
